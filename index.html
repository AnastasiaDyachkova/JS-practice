JS что же это?

Ты только начинаешь свой путь в программировании или расширяешь свои знания, познавая новые языки? Тогда сегодня мы поможем и покажем тебе один язык программирования! Возможно это поможет тебе решиться и начать изучать данные технологии. JavaScript - весьма популярный, несложный для освоения язык программирования. Был создан в 1995 году компаниями Netscape Communication Corporation и Sun Microsystems(Создателем считается Брендан Эйх). Изначально язык имел название LiveScript, так как в тот момент слово live выглядело очень привлекательным с точки зрения маркетологов.

Особенности

Одна из главных особенностей языка - мультипарадигменность. Что это значит? JavaScript поддерживает объектно-ориентированный, императивный и функциональный стили. Программы на этом языке называются скриптами. Они могут встраиваться в HTML и выполняться автоматически при загрузке веб-страницы. Скрипты распространяются и выполняются, как простой текст. Им не нужна специальная подготовка или компиляция для запуска.

Где применяется JavaScript?

JS применяется сейчас везде практически: фронтенд, бекенд, разработка приложений, игры. JavaScript не стоит на месте, и используется даже для: создания прикладного ПО для телеприставок, программирования роботов, программирования терминалов оплаты и мн.др. Основной областью применения можно считать создание интерактивных веб-страниц и приложений. Ниже, для наглядности мы покажем тебе, что ты сможешь сделать благодаря JS

S треугольника

С помощью JavaScript ты можешь решить задачи любой сложности, просто заведи необходимые переменные и создай формулу. Например здесь формула выглядит так: let s = 0.5*h*l; (h - высота, l - основание).
let здесь является директивой, объявляющей переменную с блочной областью видимости с возможностью инициализировать её значением. Если мы хотим создать неизменяемую константу, то стоит воспользоваться директивой const.

Таймер

Приступим к более сложному. Возможности языка JavaScript позволяют создать таймер. Для этого необходимо использовать методы setInterval и clearInterval. setInterval позволяет вызывать функцию регулярно, повторяя вызов через определённый интервал времени. Важно помнить, что интервал времени указывается в миллисекундах! clearInterval отменяет многократные повторения действий, установленные вызовом функции setInterval.
